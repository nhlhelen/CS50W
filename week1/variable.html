Avoid repetitiveness!
Use Scss, to create variables because normally css does not allow variables. We need to compile scss into css before the browser could use the stylesheet.

Command line: sass variables.scss:variables.css

sass --watch variables.scss:variables.css
Detects changes then update it
  1. Write scss
  2. Compile scss into css
  3. Link css to html
